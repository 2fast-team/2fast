| TutorialContentDialogHowTheAppWorks.Text                           | La funcionalidad [Autenticación] (https://es.wikipedia.org/wiki/Autenticaci%C3%B3n) del segundo factor está asegurada por la contraseña de uso único basada en la hora. Para ello, el servidor de conexión del servicio y el usuario necesitan una clave de configuración segura, que se almacena localmente en la aplicación. Esto garantiza que si la clave de configuración está activada, la aplicación podrá seguir utilizándose fuera de línea.



A diferencia de otros autenticadores, el almacenamiento de las claves de configuración para los distintos servicios de esta aplicación se realiza en un archivo, que está protegido por una contraseña desde su creación con el fin de cifrarlo. Para almacenar el archivo, se recomienda utilizar un sistema de almacenamiento en nube como Nextcloud, ownCloud, Onedrive, etc. para obtener una versión y una seguridad contra la pérdida.
